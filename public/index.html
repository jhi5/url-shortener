J. Munford's URL Shortener

How To Use

Feed the API a URL preceded by a call to /new/ such as...

http://jmunford-url-shortener.herokuapp.com/new/http://www.google.com/

And if it's a valid URL, you'll get a JSON object back with your URL and your shortcut URL. It looks something like this.

{"url":"http://www.google.com/","shortUrl":"1234"}

To use the shortened URL, just append it to the api! It looks a little something like this...

http://www.jmunford-url-shortener.herokuapp.com/1234
